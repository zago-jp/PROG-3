# 📚 Sistema de Cadastro de Livros – Java + MySQL

## 📌 Descrição
Aplicação Java orientada a objetos que realiza integração com banco de dados MySQL utilizando JDBC.  
Permite inserir e listar livros no console.

---

## 📂 Estrutura do Projeto

- Main.java  
- Livro.java  
- LivroDAO.java  
- Conexao.java  
- mysql-connector-j-9.6.0.jar  

---

## 🛠 Tecnologias Utilizadas

- Java  
- JDBC  
- MySQL  
- MySQL Connector/J  

---

## ⚙️ Execução (Windows)

Compilar:
javac -cp ".;mysql-connector-j-9.6.0.jar" *.java  

Executar:
java -cp ".;mysql-connector-j-9.6.0.jar" Main  

---

## ✅ Resultado Esperado

Exibição no console da lista de livros cadastrados no banco de dados.
